Data types at Java

The following are the data types at java :

  1. Byte: An 8-bit signed integer
  2. Short: A 16-bit signed integer
  3. Int: A 32-bit signed integer
  4. Long: A 64-bit signed integer
  5. Float: A single-precision 32-bit IEEE 754 floating point
  6. Double: A double-precision 64-bit IEEE 754 floating point
  7. Char: A single 16-bit Unicode character
